Frank Filz's Cold Iron Page

Cold Iron is a game system developed by Mark Christiansen in the 1980s while he was a student at RPI. I started running the system while a student there. The game has seen some revision since then, and I have added my own twists to it. The following are Microsoft Word documents that represent the current state of the game as I run it. These rules should all be considered Copyright Mark Christiansen and Frank Filz. Feel free to download and print them for use in my games, but I would appreciate if they are not distributed beyond my gaming group without asking permission. Note also that this material includes some material from the Talislanta and Tékumel game systems/worlds in order to support my campaigns in the Talislanta and Tékumel worlds using the Cold Iron system.
