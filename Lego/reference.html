LUGNET

LUGNET is the premier LEGO fan site. It has newsgroups, a nearly complete reference to LEGO sets (Pause), a less complete but more detailed set reference (Fibblesnork), set inventories and numerous other links including hosting an index to the Cool Lego Sites of the Week.
